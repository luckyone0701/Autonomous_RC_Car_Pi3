Autonomous RC Car for Raspberry Pi 3
------------------------------------
Features:
 - Web dashboard (http://<pi-ip>:5000)
 - 720p camera stream
 - GPS + live map
 - Manual / Autonomous toggle
 - Offline voice control (USB mic)
 - Auto start on boot

Setup:
1. Copy this folder to /home/pi/
2. Run: sudo bash install.sh
3. Access: http://<pi-ip>:5000

Repair if needed:
  ./repair.sh
